Minesweeper!

1.) Start the game by clicking on one of the squares on the grid. This reveals what's underneath the square.

2.) If the square you clicked on does not contain a mine, it will be marked with a number indicating how many mines are adjacent to it. Use this information to deduce where the mines are located.

3.) If you are certain that a square contains a mine, you can mark it by right-clicking on it.

4.) If you have marked all of the mines on the grid, you can use the left-click to reveal all of the remaining squares. If any of these squares contain mines, the game is over and you have lost.
